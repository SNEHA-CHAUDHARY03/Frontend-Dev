Responsive Principles

Fluid Layouts

Layouts that adapt to container width using % and flexible units.

width: 100%; max-width: 1200px;

Flexible Media

Images and media scale with container using max-width:100%.

img { max-width:100%; height:auto; }

Breakpoints

Use media queries to adjust layout at specific widths.

@media (max-width: 600px) { ... }
